# WaveLinux Roadmap

## Shipped

- Minimal mixer UI with channel strips, MON/STR faders, link, meter
- Stream virtual device for OBS (`WaveLinux-Stream`)
- App routing with persistence and stale-entry pruning
- Per-channel unified FX chain and preset controls
- Optional Carla launcher for VST/LV2 hosting
- Tray controls, autostart toggle, audio profile switching

## Planned

- Native in-app VST host integration
- Move peak-meter parsing off the UI thread

## Not Planned

- Stream Deck integration
- Global cross-compositor hotkey system
- Flatpak packaging (sandbox conflicts with current routing model)
