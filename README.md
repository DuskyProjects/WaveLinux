# WaveLinux 🌊
**Modern PipeWire Audio Mixer for CachyOS/KDE**

WaveLinux is a powerful, visual audio mixer designed to bring professional stream-deck style audio routing to Linux. It simplifies complex PipeWire/PulseAudio setups into a clean, intuitive interface.

## ✨ Features
- **Virtual Channel Routing**: Effortlessly split your audio into Music, Game, Browser, and System channels.
- **Dedicated Stream Mix**: Create a single virtual source for OBS that contains exactly what you want your audience to hear.
- **Container Awareness**: Native support for identifying Flatpak and Snap applications.
- **Persistence**: Remembers your routing, volume, and hardware settings between reboots.
- **AI Noise Suppression**: Built-in RNNoise support for crystal clear microphone audio.
- **Premium UI**: Designed with PyQt6 for a smooth, high-performance experience on KDE/CachyOS.

## 🚀 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/excalprimeacct-gif/WaveLinux.git
   ```
2. Run the installer:
   ```bash
   cd WaveLinux
   ./install.sh
   ```
3. Launch from your Application Menu (Start Menu).

## 🛠️ Requirements
- PipeWire & WirePlumber
- PyQt6
- pactl (PulseAudio-utils)
